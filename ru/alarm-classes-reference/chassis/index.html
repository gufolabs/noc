    Chassis¶

    Chassis | CPU | CPU Exhausted¶

    Symptoms

    Device not responce, can not establish new connections

    Probable Causes

    High CPU utilization

    Recommended Actions

    Lower storm detect threshold, filter waste traffic on connected devices, restrict SNMP Views

    Related Events

    Event Class Role
    Vendor | DLink | DxS | Chassis | CPU | Safeguard Engine enters EXHAUSTED mode opening event
    Vendor | DLink | DxS | Chassis | CPU | Safeguard Engine enters NORMAL mode closing event
